## PCB 
I used KiCad and I tried to make a compact pcb(under 10 x 10cm) with private jokes with my friends and a logo with my nickname "Cara"

<img src=pictures/pcbsch.png width="300"/>

<img src=pictures/pcb.png width="300"/>

IMPORTANT POINT !!!!!
you can see on the electrical sketch that I used some Leds but they do not appear on the PCB that's normal
I added an output for the LEDs below the encoder. I want to use the Leds around the pcb to make an effect that diffuse the light with a fit layer of PLA
So the LEDs should not appear on the PCB.

## BOM

Here's the list of parts for my project!

 - 1x Seeed XIAO RP2040
 - 6x Through-hole 1N4148 Diodes
 - 6x MX-Style switches
 - 1x EC11 Rotary encoders
 - 1x 0.91 inch OLED displays
 - 6x Blank DSA keycaps (White)
 - 14x SK6812 MINI-E LEDs
 - 4x M3x16mm screws
 - 4x M3x5mx4mm heatset inserts
 - 1x 3D printed case: not shure about the color
 - 1x PCB
